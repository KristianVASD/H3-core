# H3 Basic Shell

The Basic Shell is a lightweight open tool to:

* validate H3 data
* inspect building datasets
* load `.h3pkg` files

---

## 🧰 Features

* JSON validation
* Manifest parsing
* Simple data viewer (HTML)

---

## 🚀 CLI Usage

### Validate a package

```bash
h3 validate path/to/package.h3pkg
```

### Validate a manifest

```bash
h3 validate manifest/minimal_manifest.json
```

---

## 📊 Demo Viewer

Open:

```bash
index.html
```

This will:

* load example data
* show assets and events
* demonstrate structure

---

## 🎯 Purpose

The Basic Shell is not meant to be powerful.

It exists to:

* prove the standard works
* enable adoption
* give developers a starting point
